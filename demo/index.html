SPA navigation
for static sites

Fiamma intercepts link clicks, fetches the next page, and swaps the DOM. No full reload, no framework.

~2kb minified. No dependencies.

import { Router, EnableProgressBar } from 'fiamma';

EnableProgressBar();
Router();

What it does

Clicks on same-origin links are caught before the browser navigates. The next page is fetched in the background, the <head> is diffed, and the <body> is swapped. History is updated, scroll resets, scripts re-run.

Add fiamma-preserve to any element and Fiamma keeps that DOM node alive across navigations. The navbar above uses it.

Try it

Navigate between the pages using the links above. The navbar never re-renders — watch the indicator slide. The page content fades in. The progress bar appears on slower connections.
